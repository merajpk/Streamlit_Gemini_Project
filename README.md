# Gemini API Note Summarizer

__Project Overview__

This project is a **Python-based Note Summarizer application** powered by the **Google Gemini API**. It helps users transform long notes into concise summaries and also provides additional learning features like quizzes and audio output.

__Features__

-Text Summarization** – Convert long notes into short, meaningful summaries
-AI Quiz Generator** – Automatically generate quiz questions from notes
-Audio Output** – Convert summarized text into speech
-Powered by Gemini API** – Uses Google Gemini for intelligent text processing


__Tech Stack__

* Python 
* Google Gemini API
* Streamlit
* Text-to-Speech (TTS) library
* Other Python libraries (as required)
